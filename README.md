# Three Tasks Support

This repository contains the support website for Three Tasks, a minimalist task manager that limits you to 3 tasks per day.

## Overview

The support website provides:
- Contact information for user support
- Privacy policy documentation
- General app information

## Website Structure

- `index.html` - Main support page with contact information
- `privacy-policy.html` - Privacy policy for the Three Tasks app
- `.gitignore` - Git ignore file for common files and directories
